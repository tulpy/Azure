# PowerShell sample for remediating "Enable transparent data encryption on SQL databases"
This sample PowerShell script remediates the "*Enable transparent data encryption on SQL databases*" recommendation in Azure Security Center.<br><br>

Provide your Sql Server name, Azure resourcegroup name and your database name.<br>
The sample shows how you can get the TDE status and how to set it.<br><br>

Please ensure that you have installed the PowerShell Az module, which can be downloaded by executing the following in a PowerShell session:
```powershell
Install-Module -Name "Az" - verbose
```
